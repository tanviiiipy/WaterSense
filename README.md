# WaterSense
WaterSense is a Streamlit-based smart water management dashboard designed to promote water conservation through monitoring, analytics, and gamification. Users can add virtual sensors, visualize water consumption trends, detect potential leaks, earn achievements, participate in water-saving challenges, and explore interactive reports.

Features:
Sensor management system
Water usage dashboard and analytics
Leak detection simulation
Gamification and achievements
Water-saving challenges
Notifications and alerts
Community leaderboard
Data import/export support
Interactive water conservation game

Tech Stack: 
Python
Streamlit
Pandas
NumPy
